## Instalación ⬇️

  Clona el proyecto

  ```bash
    git clone 
  ```

  Go to the project directory

  ```bash
    cd my-project
  ```

  Obtén las dependencias del pubsec.yaml

  ```bash
    flutter pub get
  ```

  Inicia el servidor

  ```bash
    flutter run
  ```
